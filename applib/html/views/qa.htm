Today's Compliance Question

A collector calls the Consumer's cell phone using an Automatic Telephone Dialing System (ATDS). The Consumer tells the Collector he can't call his cell phone anymore. The next day, the Collector manually dials the Consumer's cell phone from a non-Automatic Telephone Dialing System (ATDS). What is the Collector's TCPA liability?

  1. Possibly $1500
  2. Definitely $1500
  3. Definitely $500
  4. Possibly $500

Feedback

We don't know whether the Collector originally had consent to call the Consumer, so we don't know if the first call was a violation. The second call would have been a violation if the Collactor was using an Automatic Telephone Dialing System (ATDS) because he knowingly called without consent.
