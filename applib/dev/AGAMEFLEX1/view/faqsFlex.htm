Scoring/Points

Avg. Points Per Player, Avg. Score

Avg. Points Per Player: The average points that each player is contributing to the total points scored for a game. Points can be scored on a daily basis, so it it possible that a single individual can average more than the maximum points possible for a single day. This calculation is determined by the current points total (Points) for the game divided by Number of Players on the team (# of Players).

Avg. Points Per Player = Points / # of Players

Avg. Score: The average score for the players that have any score for the scoring metric.
